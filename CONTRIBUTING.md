# Contributing to MyMCPApp

Thank you for contributing! We welcome contributions from new authors. Follow these steps to add a new post:

1. Fork the repository and clone your fork.
2. Create a new branch named `add-post-<short-topic>`.
3. Add a Markdown file under `posts/` (e.g. `posts/new-feature.md`).
4. Include front matter at the top:
   ```yaml
   ---
   title: "Your post title"
   date: YYYY-MM-DD
   author: "Your Name"
   ---
   ```
5. Write content in Markdown in the body section.
6. Run app checks (if any) or local preview (e.g. `npm start`).
7. Commit with a clear message:
   `git commit -am "Add post: Your title"`
8. Push your branch:
   `git push origin add-contributing-guidelines` (or your branch name).
9. Open a pull request describing what you changed and why.

We review contributions promptly and may ask for small edits. Thanks for helping improve MyMCPApp!
